# StudyFlow

StudyFlow är en JavaFx-applikation utvecklas i Java med Maven.

## Funktioner (planerade)
- Skapa kurser
- Lägga till uppgifter
- Kalender
- Statistik över studietid
- Påminnelser
- Mörkt läge

## Teknik

- Java 21
- JavaFx 21
- Maven
- Git
- IntelliJ IDEA

## Köra projektet
````bash
  mvn javafx:run
````
## Projektstruktur

```
src/
├── main/
 │   ├── java/
 │   └── resources/
 └── test/
```
